# node-rest-example

An example rest api with NodeJS and MongoDB

## Prerequisite

This REST API example assumes that you have NodeJS and MongoDB installed.

If not, install them and proceed.

## Usage

```sh
npm i # or yarn

npm run start # or yarn start
```
